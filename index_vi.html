Tổng quan

Giải pháp toàn diện cho việc tạo ra hình ảnh PNG trong suốt

Dự án này triển khai một giải pháp toàn diện để huấn luyện và tạo ra hình ảnh PNG trong suốt không có nền sử dụng mô hình Flux.1 dev. Việc triển khai dựa trên kiến trúc của dự án Microsoft ART-MSRA nhưng được điều chỉnh đặc biệt cho việc tạo ra hình ảnh PNG trong suốt một lớp.

Giải pháp bao gồm các thành phần sau:

  • Module huấn luyện: Huấn luyện VAE đã được điều chỉnh trên hình ảnh PNG trong suốt
  • Module lưu trữ: Lưu các mô hình VAE và decoder đã huấn luyện
  • Module tạo hình ảnh: Tạo ra hình ảnh PNG trong suốt mới
  • Tích hợp ComfyUI: Các node tùy chỉnh để sử dụng các mô hình đã huấn luyện trong ComfyUI

Pipeline huấn luyện

Huấn luyện mô hình VAE trên hình ảnh PNG trong suốt từ Google Drive của bạn.

  • Tự động kết nối với Google Drive của bạn
  • Hiển thị tiến trình huấn luyện
  • Lưu các mô hình đã huấn luyện để sử dụng sau này

Tạo hình ảnh

  • Tạo hình ảnh từ văn bản với độ trong suốt
  • Tạo hàng loạt với nhiều prompt
  • Công cụ thử nghiệm tham số

Hướng dẫn sử dụng

Cách sử dụng trình tạo PNG trong suốt

Sử dụng với Google Colab

  1. Mở Notebook huấn luyện

    Nhấp vào nút "Mở trong Colab" cho notebook Pipeline huấn luyện.

  2. Kết nối với Google Drive

    Chạy ô đầu tiên để kết nối với Google Drive của bạn.

  3. Cấu hình đường dẫn

    Đặt đường dẫn cho dữ liệu huấn luyện, đầu ra mô hình và hình ảnh được tạo ra.

    DATA_DIR = "/content/drive/MyDrive/SD-Data/TrainData/4000_PNG/TEST"
OUTPUT_DIR = "/content/drive/MyDrive/VAE-DECODER"
  4. Huấn luyện mô hình

    Chạy các ô huấn luyện để huấn luyện mô hình VAE của bạn.

  5. Tạo hình ảnh

    Mở notebook Tạo hình ảnh và sử dụng mô hình đã huấn luyện của bạn để tạo PNG trong suốt.

Cài đặt cục bộ

  1. Clone repository 
    git clone https://github.com/your-username/flux-transparent-png.git
cd flux-transparent-png
  2. Cài đặt dependencies 
    cd python
python install.py
  3. Huấn luyện mô hình 
    python main.py train \
  --data_dir "/path/to/transparent/pngs" \
  --output_dir "/path/to/save/models" \
  --batch_size 8 \
  --num_epochs 100
  4. Lưu các mô hình 
    python main.py save \
  --checkpoint_path "/path/to/checkpoint.pt" \
  --output_dir "/path/to/save/models"
  5. Tạo hình ảnh 
    python main.py generate \
  --model_path "/path/to/model.pt" \
  --prompt "Một bông hoa đẹp trên nền trong suốt" \
  --output_dir "/path/to/save/images"

Tích hợp ComfyUI

  1. Cài đặt ComfyUI

    Tải xuống và cài đặt ComfyUI theo hướng dẫn trên trang GitHub của họ.

  2. Cài đặt các node tùy chỉnh

    Sao chép file comfyui_transparent_png_nodes.py vào thư mục custom_nodes trong cài đặt ComfyUI của bạn.

  3. Cài đặt các mô hình

    Sao chép các mô hình VAE và decoder đã huấn luyện của bạn vào thư mục models/transparent_vae trong cài đặt ComfyUI của bạn.

  4. Sử dụng các node

    Khởi động ComfyUI và sử dụng các node tùy chỉnh sau:

    • Load Transparent VAE: Tải mô hình VAE trong suốt đã huấn luyện
    • Generate Transparent PNG: Tạo hình ảnh trong suốt từ prompt
    • Save Transparent PNG: Lưu hình ảnh dưới dạng PNG trong suốt
