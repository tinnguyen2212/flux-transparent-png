Flux.1 Transparent PNG Generator

Train and generate transparent PNG images using Flux.1 dev

Overview

This project implements a comprehensive solution for training and generating transparent PNG images without backgrounds using the Flux.1 dev model. The implementation is based on the Microsoft ART-MSRA project architecture but adapted specifically for single-layer transparent PNG generation.

The solution consists of the following components:

  • Training Module: Trains a modified VAE on transparent PNG images
  • Saving Module: Saves the trained VAE and decoder models
  • Generation Module: Generates new transparent PNG images
  • ComfyUI Integration: Custom nodes for using the trained models in ComfyUI

Quick Start Guide
1. Clone the repository
git clone https://github.com/your-username/flux-transparent-png.git
cd flux-transparent-png
2. Install dependencies
python install.py
3. Train the model
python main.py train \
  --data_dir "/content/drive/MyDrive/SD-Data/TrainData/4000_PNG/TEST" \
  --output_dir "/content/drive/MyDrive/VAE-DECODER" \
  --batch_size 8 \
  --num_epochs 100
4. Save the trained models
python main.py save \
  --checkpoint_path "/content/drive/MyDrive/VAE-DECODER/checkpoints/checkpoint_epoch_100.pt" \
  --output_dir "/content/drive/MyDrive/VAE-DECODER"
5. Generate transparent PNG images
python main.py generate \
  --model_path "/content/drive/MyDrive/VAE-DECODER/transparent_vae.pt" \
  --prompt "A beautiful flower on a transparent background" \
  --output_dir "/content/drive/MyDrive/VAE-DECODER/OUT"
